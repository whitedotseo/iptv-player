# IPTV Player

Nowoczesny odtwarzacz IPTV dla Windows z obsługą Xtream Codes API.

## Pobieranie

👉 **[Kliknij tutaj aby pobrać IPTVPlayer.exe](../../releases/latest)**

## Funkcje
- Live TV z EPG
- Filmy i Seriale (VoD)
- Nagrywanie i planifier
- Siatka okładek
- Archiwum (Catchup)
- Historia oglądania
- Kontrola rodzicielska
- Filtr gatunków
- Skaner języków (PL audio/napisy)

## Wymagania
- Windows 10/11 (64-bit)
- Połączenie z serwerem Xtream Codes
